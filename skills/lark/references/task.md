# 任务

管理任务和清单：创建待办、查看/更新状态、拆分子任务、组织清单、分配协作成员。

| Shortcut | 说明 | 详细文档 |
|----------|------|---------|
| `+create` | 创建待办任务 | [lark-task-create.md](lark-task-create.md) |
| `+update` | 更新任务信息 | [lark-task-update.md](lark-task-update.md) |
| `+comment` | 添加任务评论 | [lark-task-comment.md](lark-task-comment.md) |
| `+complete` | 完成任务 | [lark-task-complete.md](lark-task-complete.md) |
| `+reopen` | 重新打开任务 | [lark-task-reopen.md](lark-task-reopen.md) |
| `+assign` | 分配负责人 | [lark-task-assign.md](lark-task-assign.md) |
| `+followers` | 管理关注者 | [lark-task-followers.md](lark-task-followers.md) |
| `+reminder` | 设置提醒 | [lark-task-reminder.md](lark-task-reminder.md) |
| `+get-my-tasks` | 获取我的任务列表 | [lark-task-get-my-tasks.md](lark-task-get-my-tasks.md) |
| `+tasklist-create` | 创建任务清单 | [lark-task-tasklist-create.md](lark-task-tasklist-create.md) |
